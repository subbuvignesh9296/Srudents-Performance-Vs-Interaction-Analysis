# **📊 Screen Time vs Sleep Hours Analysis Dashboard**
# **📌 Project Overview**
* This project is created with power bi dashboard mainly focuses on students performance depends on the screen time Vs sleep hours.
* This Dashboard provide the information about how students liftstyle has been changed due to screen time and how their performance was afftect by usage of screen time.
* The project uses interactive visualizations, KPI cards, charts, and analysis reports to present data in a simple and understandable format.

# **🎯 Objectives**
* Analyse total sleep time and Screen hours
* Identify trends and patterns between screen usage and sleep duration
* Provide insights for healthier digital lifestyle decisions

# **🛠️ Tools & Technologies Used**
* Power BI – Dashboard creation and data visualization
* Microsoft Excel / Dataset – Data source and preprocessing
* Data Analysis Techniques – Trend analysis and comparison.

# **📈 Dashboard Features**
* KPI
  * Total attandance
  * Overall Percentage
  * Overall Average
  * Analytical Score
  * Result

# **🔹 Visualizations**
* **Pie Chart** - Result of students by parents Education
* **Donut Chaart** - Final scores of the students by their attendance and studies.
* **Bar Chart** - Overall percentange by the result.
* **Column Chart** - Screen time analysis.
* **Ribbon Chart** - Total Comaprision.

# **🔹 Insights Section**
* Understanding the effect of excessive screen usage.
* Identifying balanced and unhealthy patterns.
* Comparing user behavior trends.

# **📊 Key Insights**
* Increased screen time may reduce average sleep duration.
* Users with balanced screen usage tend to maintain healthier sleep patterns.
* Visual analysis helps identify trends and behavioral differences quickly.
* Interactive charts make data exploration easier for decision-making.

# **💡 Recommendations**
* Reduce screen exposure before bedtime.
* Maintain a healthy balance between digital activity and sleep.
* Use data-driven monitoring to improve lifestyle habits.
* Encourage awareness of healthy screen time practices.
